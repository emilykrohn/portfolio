Waddle

Godot Game Jam

Waddle is a game created with Godot using gdscript for the second game jam I participated. The player controls a penguin that collects falling presents and sorts them by color. The sleigh in the background will show the player’s progress by increasing the amount of presents in it. I created the game sprites using Aseprite.

The presents spawn in a random location at the top of the screen and when they reach the players hit box the penguin will be shown holding the present. The player can move up, down, left and right.

Main Menu

Game Screen

Win Screen

The main menu is shown when the game is first opened. The player collects the presents and then presses the space bar next to the correct box at the bottom of the screen. The player will be shown the win screen when the sleigh in the background is full and they will be given the option to restart the game.
